Three Tarts Bakery Redesign

To redesign the site of my previous employer to something a bit more conventional or expected design for a bakery.

Goals

  1. Create a home page that is visually stimulating and easy to use. One that has a place for specials, and a link other than the nav bar to bring you to them. In addition there is a space announcements for upcoming events and menu changes.
  2. Create a page that has information on how to order a cake with the options for said cake in easy to use drop down menus or bullet point clicks. This will utilize images and words to do so.
  3. Make a menu page for the cafe’s menu with a rotating specials page and a regular menu. This will be done similarly to how the hewn website handles their menu and specials.

Site Model: Hewn Bakery Site

I chose this site due to it’s more user friendly interface. The nav bar above is one line with a logo in the middle with the opening photo being the first things a customer would see when opening the site. In addition there are links that show right off the bad some menu and special items that can bring you straight there.

Site Pages

  1. Home Page

    This page will include a navigation bar, as well as the logo in the header. The page should have a large welcome picture, a place for news and some of that days specials. The footer should contain the location and hours of the bakery. In addition there should be links to socials.

  2. Cake Order Form Page

    This page should contain all information on how to order a cake and how to customize it. One should have options for all cakes that exist and how big, then options for custom cakes. Then should have options for finish, decorations and writing to name a few. Pictures will be provided if possible and should be similar to the menu page at hewn with hovering images.

  3. Menu Page

    Like the menu page at hewn there should be a place for specials first, then regular menu. It will be organized with pictures that when hovered over will show the name of the item,(if possible if I can’t get enough images, I’ll just do what I can).

Target Audience

Those in the local community who are in need of a birthday or special occasion cake, or are simply wishing for a sandwich or salad for lunch.
